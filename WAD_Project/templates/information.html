Information :

In this HelpingHand Users are categorized into 2 types: Donors
                                                        Receivers
  
  User can Login using his/her Username,phone number and also via Google & Social media accounts.
  Pages : Home  | Receive | Donate  | Chat |
  
  Home Page : At bottom of the home page, there are 2 options :
                                                                Donate  &  Receive.
  
  
  Receive : Browse by Categories: Select the desired thing.
                   Each item is displayed in list format showing {title} When clicked on item:
                   Opens a new page showing option for browsing for donors : Location wise and Category wise.
    
  Categories :  Food, clothes, Accessories, Footwear, gadgets, books, Sport items and others
  
  Chat : when receiver selects a donor,end to end chat option is provided between donor and receiver.
  
  Donate : A webpage showing options for uploading an image of item,title and address to collect.
                 There will be options of naming his/her state,district and the collecting address.
